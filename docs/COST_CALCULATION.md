## 3. 플랜별 비용 계산

### Free 플랜 (무료)
**제한사항:**
- AI 채팅: 일 5회
- 학습 기능: 주 1회
- 트렌드 브리핑: 기본 5개
- 주간 리포트: 없음

**월간 예상 비용 (사용자당):**
```
일일 비용:
- AI 채팅 (5회): 2,000 * 5 * 0.00015 + 500 * 5 * 0.0006 = $0.0015 + $0.0015 = $0.003
- 모닝 브리핑: 1,500 * 0.0000025 + 800 * 0.00001 = $0.00375 + $0.008 = $0.012
- 일정 추천 (1회): 1,000 * 0.00015 + 400 * 0.0006 = $0.00039

주간 비용:
- 학습 슬라이드 (1회): 5,000 * 0.0000025 + 3,000 * 0.00001 = $0.0425
- 트렌드 브리핑: 공유 캐시 사용 (무료)

월간 총계: (0.003 + 0.012 + 0.00039) * 30 + 0.0425 * 4 = $0.46 + $0.17 = $0.63/월
```

### Max 플랜 ($9.99/월)
**기능:**
- AI 채팅: 무제한
- 학습 기능: 무제한
- 개인화 트렌드 브리핑
- 주간 리포트
- RAG (과거 대화 컨텍스트)
- 자비스 모드 (고급 분석)

**월간 예상 비용 (사용자당):**
```
일일 비용:
- AI 채팅 (20회): 2,000 * 20 * 0.00015 + 500 * 20 * 0.0006 = $0.006 + $0.006 = $0.012
- 모닝 브리핑 (개인화): 1,500 * 0.0000025 + 800 * 0.00001 = $0.012
- 일정 추천 (5회): 1,000 * 5 * 0.00015 + 400 * 5 * 0.0006 = $0.002
- 하루 요약: 1,500 * 0.0000025 + 600 * 0.00001 = $0.01

주간 비용:
- 학습 슬라이드 (5회): 5,000 * 5 * 0.0000025 + 3,000 * 5 * 0.00001 = $0.21
- 퀴즈 생성 (5회): 3,000 * 5 * 0.0000025 + 2,000 * 5 * 0.00001 = $0.14
- 퀴즈 채점 (5회): 2,000 * 5 * 0.0000025 + 1,000 * 5 * 0.00001 = $0.075
- 주간 피드백: 3,000 * 0.0000025 + 1,500 * 0.00001 = $0.0225
- 습관 분석: 2,000 * 0.00015 + 800 * 0.0006 = $0.0008

월간 총계: (0.012 + 0.012 + 0.002 + 0.01) * 30 + (0.21 + 0.14 + 0.075 + 0.0225 + 0.0008) * 4
         = $1.08 + $1.79 = $2.87/월
```
